IT238 Final Project Summary

For this project, we decided to take the archeology game board and up it up a notch. We wanted to add a drag and drop feature to the game. To do this,
we added a image of a shovel and gave it the ability to have an actionable drag and drop feature to make it seem you're really digging up spots looking for these artifacts.




What the project does:

- This final project builds upon the original assignment 4 for this class.

- We implemented a drag on drop feature that was previously talked about in class, but not used in any of the assignments

- Instead of the user clicking on a square on the grid to search for an artifact, we removed that ability and made it where the only way to reveal a square, is to drag and drop the shovel over the desired location.

- We have a table indicating how many of each artifact is left to be found with corresponding images for easy readability. There will also be message showing when you have found a piece or found the entirety of an artifact.

- Once the game is completed, an alert will pop up letting you know so. It will also give you a rating on how you did. Excellent, good, or terrible.

- We also added a header menu that doesn’t link to anything really, other than the summary page under the “about the project” list. We just wanted to show that we could make this possible.

- We also added hard mode that makes the board 14x14. You can change the difficulty back and forth whenever you please. It will reset the board and score upon changing difficulties.

- We had a little fun with the CSS and tried experimenting with different texts and effects very briefly. We stuck with a glow effect for the dig/artifact counter above the game board.
